## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/felipebsa/cvvj.git
cd cvvj/backend

# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Run the server
uvicorn main:app --reload
```
